## laravel-aliyun-sms-service

 - A Laravel Plugin For Encapsulation AliYun SMS Services.

----------

### Requirement

 - PHP >= 7.4
 - laravel >= 6.0
 - illuminate/support
 - overtrue/easy-sms

----------

### Installation

 - installing plugins package

```shell
composer require "zxf5115/laravel-aliyun-sms-service"
```
